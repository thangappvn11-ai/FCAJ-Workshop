### BƯỚC 1: Đổi tên Repository (hoặc Tạo mới) trên GitHub.com

1. Đăng nhập [GitHub.com](https://github.com).
2. Nếu bạn đã tạo repo trước đó:
   - Mở repository đó -> vào **Settings** -> Mục **Repository name** đổi tên thành: `thang2308.github.io` -> Bấm **Rename**.
3. Nếu chưa tạo:
   - Bấm dấu **+** (góc trên bên phải) -> Chọn **New repository**.
   - **Repository name**: Đặt tên chính xác là: `thang2308.github.io`
   - Đặt là **Public** -> Bấm **Create repository**.

---

### BƯỚC 2: Upload code lên GitHub

**Cách A: Upload trực tiếp trên Web (Nhanh nhất)**
1. Mở repo `thang2308.github.io` trên GitHub.
2. Bấm **Add file** -> Chọn **Upload files**.
3. Chọn toàn bộ file/thư mục trong `d:\VietThang.github.io-main` kéo thả vào trang web.
4. Kéo xuống bấm **Commit changes**.

**Cách B: Dùng phần mềm GitHub Desktop**
1. Tải [GitHub Desktop](https://desktop.github.com/).
2. Đăng nhập nick `thang2308`.
3. Vào **File** -> **Add local repository** -> Chọn thư mục `d:\VietThang.github.io-main`.
4. Bấm **Publish repository** với tên `thang2308.github.io`.

---

### BƯỚC 3: Kích hoạt GitHub Pages (Để hết lỗi 404)

1. Mở repo `thang2308.github.io` trên GitHub.
2. Vào **Settings** -> chọn **Pages** ở danh sách bên trái.
3. Mục **Build and deployment** > **Source**:
   - Chọn **Deploy from a branch**.
   - Mục **Branch**: Chọn `gh-pages` (hoặc `main`) và folder `/ (root)` -> Bấm **Save**.
4. Chờ khoảng 1 - 2 phút, mở lại trang:
   👉 **https://thang2308.github.io/**
