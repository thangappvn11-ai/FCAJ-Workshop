#### Lab Introduction

This workshop focuses on automating the entire deployment and operation lifecycle of the VDCMS (Voice-Driven Construction Management System) – a smart progress and safety management solution for construction sites powered by intelligent speech recognition technology. The system addresses a real-world problem at construction sites: enabling field engineers to generate reports quickly using voice commands, which are automatically transcribed into text and synchronized to the Cloud for immediate managerial approval.

#### Technical Objectives

* Successfully validate secure email delivery protocol configurations on the Cloud using Amazon SES.
* Implement Infrastructure as Code (IaC) principles to deploy complex resource stacks automatically and synchronously via AWS CloudFormation, rather than configuring individual services manually.
* Distribute a serverless Frontend application with high performance and low latency through the integration of Amazon S3 and Amazon CloudFront.
* Evaluate the integrity of a multi-tier Cloud Production architecture, including: Backend application servers (EC2), Relational Databases (Aurora RDS), User Session Identity Management (Cognito), Encrypted Secrets Management (Secrets Manager), Asynchronous Message Queues (SQS), and Web Application Firewalls (WAF).
