* **Account permissions:** Use an AWS account with root-level permissions (`AdministratorAccess`) or authorized administrator rights to ensure you can create, modify, and delete the Stack without being blocked by IAM policies.
* **CloudShell environment:** A browser-based command-line tool with the latest AWS CLI, Git, Node.js, and NPM pre-installed to install dependencies.
* **Configuration information:**
  * Project source code repository: [https://github.com/BuiThanhPhuoc/Voice-Driven-Construction-Management-System.git](https://github.com/BuiThanhPhuoc/Voice-Driven-Construction-Management-System.git)
  * Personal email used as the notification send/receive verification target: **admin.vdcms@gmail.com** (also prepare a secondary email to test new account registration: **engineer.vdcms@gmail.com**)
